# 五子棋游戏 (WZQ-Game)

一个基于 HTML5 Canvas 开发的五子棋游戏，支持双人对战，提供完整的游戏功能和优美的用户界面。

## 功能特性

- 🎮 **双人对战** - 支持黑白棋交替落子
- 🏆 **胜负判定** - 智能判断横、竖、斜向五子连线
- 🎨 **精美界面** - 现代化 UI 设计，响应式布局
- 🔊 **音效反馈** - 落子、胜利、悔棋等音效
- ⌨️ **键盘支持** - 快捷键操作
- 📱 **移动端适配** - 支持触摸操作

## 游戏操作

### 鼠标操作
- 点击棋盘格点落子
- 鼠标悬停显示落子提示

### 键盘快捷键
- `R` - 重新开始游戏
- `Ctrl + Z` - 悔棋
- `Esc` - 重新开始游戏

### 控制按钮
- **重新开始** - 重置游戏
- **悔棋** - 撤销上一步操作

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **图形渲染**: Canvas API
- **架构**: 模块化设计，MVC 模式
